# 🚀 Factory Reallocation & Shipping Optimization Dashboard

An enterprise-grade **Decision Intelligence Dashboard** for optimizing factory allocation, transport costs, and supply chain performance using machine learning, SHAP explainability, and real-time analytics.

---

## 📌 Overview

This project is designed to simulate and optimize supply chain operations for a distribution network. It provides actionable insights into factory performance, transport efficiency, and operational risks.

The dashboard helps decision-makers:

- Identify bottlenecks in factory operations  
- Reduce transport costs and lead times  
- Optimize factory allocation strategies  
- Monitor key performance indicators (KPIs)  
- Make data-driven decisions using predictive analytics  

---

## ⚙️ Key Features

### 📊 Analytics & Insights
- Factory performance monitoring  
- Transport cost analysis  
- Lead time prediction using ML models  
- Region-wise and factory-wise analysis  
- KPI tracking (cost, lead time, efficiency)

### 🤖 Machine Learning
- Predictive modeling for lead time  
- Model evaluation and comparison  
- Confidence interval estimation  
- SHAP explainability for model insights  

### 🔁 Simulation & Optimization
- Factory reassignment simulator  
- Scenario-based decision analysis  
- What-if analysis for operational improvements  

### 🚨 Decision Intelligence
- Risk detection and alert system  
- Recommended actions for optimization  
- Executive decision summary  
- Performance trends and insights  

### 💬 Interactive UI
- Clean and modern dashboard layout  
- Interactive charts and visualizations  
- Section-based navigation  
- Real-time data exploration  

---

## 🛠️ Tech Stack

- **Frontend/UI**: Streamlit  
- **Backend**: Python  
- **Data Processing**: Pandas, NumPy  
- **Machine Learning**: Scikit-learn  
- **Visualization**: Plotly, Matplotlib  
- **Explainability**: SHAP
